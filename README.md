# Projeto-BI---Venda-de-Albuns
Este dashboard foi desenvolvido no Power BI para analisar os fatores de sucesso e a concentração de mercado dos álbuns musicais mais vendidos de todos os tempos. O objetivo é transformar dados brutos em insights estratégicos sobre a indústria fonográfica, explorando tendências temporais e a dominância de gêneros e artistas.
